Detailed Description

template<int N>
class Kalmar::index< N >

Represents a unique position in N-dimensional space.

Template Parameters
NThe dimensionality space into which this index applies. Special constructors are supplied for the cases where , but N can be any integer greater than 0.

Default constructor.

The value at each dimension is initialized to zero. Thus, "index<3> ix;" initializes the variable to the position (0,0,0).
